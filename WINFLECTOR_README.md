# Winflector 4.0.3.4 README

Winflector provides thin-client remote access to Windows applications across platforms like Windows, Linux, Android, macOS, and Raspberry Pi. The free version limits to two concurrent users and supports commercial use without time restrictions.[1][2]

## Quick Start
1. Download the latest installers from winflector.com (e.g., Server/Client package for Windows).[3][4]
2. Install server on the host machine with apps to share; open TCP port 6601 in firewall.[3][2]
3. Configure users and enable app sharing in server settings.[2]
4. Install client on your device and connect via IP/hostname, username, password, app path.[3]
5. Launch: Apps run on server; screen displays on client.[4][5]

## New in 4.0.3.4 (Dec 22, 2025)
- Faster client auto-updates.[6]
- Enhanced remote desktop admin tool.[6][7]
- File transfer: Ctrl+Alt+G (get from server), Ctrl+Alt+P (put to server, saves to C:\Windows\Temp).[7]
- Drag-and-drop text support; configurable strong passwords; URL-based resolution.[7]
- Preps for universal macOS client (Intel/ARM).[6]
